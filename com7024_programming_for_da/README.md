# Housing in Berlin
Performing Exploratory Data Analysis on a dataset containing residential property information of Berlin from 1900 to 2015.

## Requirements
- If working locally:
	- Jupyter Notebook
	- Python3.10+ and required libraries (Pandas, Numpy, Scipy, Matplotlib, and Seaborn)
- If working online:
	- A Jupyter Notebook instance like Google Colab
- The dataset (found at the root of this directory)
